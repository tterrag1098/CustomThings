CustomThings [![Build Status](http://ci.tterrag.com/job/CustomThings/badge/icon)](http://ci.tterrag.com/job/CustomThings/)

You can read documentation at [**the wiki**](https://github.com/tterrag1098/CustomThings/wiki)
